Thanks for trying custom functions for Excel! You've installed the sample functions, which start with the prefix CONTOSO. Try typing them into a cell (with or without this task pane - it isn't needed):
  • =CONTOSO.ADD42 adds 42 to two input numbers.
  • =CONTOSO.ADD42ASYNC does the same thing but waits 1 second first.
  • =CONTOSO.ISPRIME checks whether the number is prime and returns true or false.
  • =CONTOSO.NTHPRIME returns the Nth prime. Watch out - it needs a lot of calculation if N is big!
  • =CONTOSO.GETDAY returns the current day of the week as a string.
  • =CONTOSO.SECONDHIGHEST lets you select a range of data and finds the second highest number.
To change the functions, edit customfunctions.js and customfunctions.json. To change the prefix, edit customfunctions.xml. For more information, see https://aka.ms/customfunctions.
